Buttons

Colours

Buttons must be given a class of .button in order for the default styles to be applied.

Otherwise, a class of .button_* with the theme name will create a button with the specified colour. E.G.: .button_primary.

				<div class="col">
				<button class="button button_secondary">Button</button>
				</div>

Cards

The purpose of cards are to display site infomation in a the style of a content container. These could include text, icons, charts, images, progress indicators, or any other general information that a user may need to know. A c ard can have a header, body or a footer and are similar to that of content boxes or panels.
